# Latimore OS Research → Content Loop

Use this skill for carrier PDFs, product training, estate planning docs, competitor notes, Chamber/local market material, or client education assets.

## Workflow

1. Ingest source files and extract the core claims.
2. Separate source-grounded facts from marketing interpretation.
3. Generate one or more outputs:
   - client education post
   - Google Business Profile post
   - sales script
   - follow-up sequence
   - training notes
   - presentation outline
   - FAQ
4. Keep language plain-English unless a technical audience is specified.
5. Add compliance notes where claims involve insurance, annuities, indexed strategies, living benefits, or guarantees.

## NotebookLM usage pattern

When external NotebookLM tooling is available, upload source files into a notebook, ask source-grounded questions, then generate derivative assets. Do not treat generated artifacts as authoritative without checking the source text.
